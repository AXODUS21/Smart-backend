# Role: Admin
**Goal**: Platform operations and user support.

### Core Workflow
- **User Management**: Oversee Students, Tutors, and Principals. Handle account issues and manual adjustments.
- **Tutor Onboarding**: Review and approve Tutor applications. Verify qualifications and set regional pay rates.
- **Policy Config**: Adjust platform-wide booking policies, cancellation rules, and session limits via the "Settings" tab.
- **Voucher Processing**: Process requests for manual voucher/credit assignments.
- **Announcements**: Broadcast platform-wide messages to users.
- **Subject Management**: Manage the list of subjects available for tutoring.
- **CMS Management**: Update blog posts, services, "Why Choose Us" sections, and FAQs.
