# Role: Superadmin
**Goal**: Global system control and strategic oversight.

### Core Workflow
- **Global Analytics**: High-level reporting on revenue, expenses, and platform growth metrics.
- **Sidebar Configuration**: Customize the dashboard navigation menu for all admin users based on their specific responsibilities.
- **Task Assignment**: Assign operational tasks to specific Admins and monitor their completion.
- **Batch Withdrawals**: Finalize and process tutor payout batches in bulk.
- **System Settings**: Manage global templates (e.g., lesson report documents) and core security protocols.
- **Admin Management**: Create and manage Admin accounts and their permissions.
