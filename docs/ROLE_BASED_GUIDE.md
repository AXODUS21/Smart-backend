# Role-Based User Guide: Smart Tutoring Platform

This directory provides tailored instructions for each of the five user roles on the platform. Click on a role to view its dedicated guide.

## Available User Roles

- [**Student**](roles/student.md)
  - Booking, managing credits, and attending sessions.
- [**Tutor**](roles/tutor.md)
  - Availability management, accepting requests, and earnings.
- [**Principal (School Admin)**](roles/principal.md)
  - School management and institutional oversight.
- [**Admin**](roles/admin.md)
  - Platform operations, approvals, and user support.
- [**Superadmin**](roles/superadmin.md)
  - Global system control, analytics, and strategy.
